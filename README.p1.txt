In this part We are given a Java class that represents  person one and modifies a string by rotating its characters by 2 positions.
To modify the Person1 class so that it takes user input for the string to be modified, we have incorporate the Scanner class in Java.
This program effectively demonstrates how user input can be leveraged to create a personalized experience by combining a user's name
with a dynamically modified string.
